## JavaScript Array Methods

This project was made in order to recreate the most common array methods used in JavaScript.

I've created my own implementation of this methods to test my knowledge.

Which includes:

- foreach()
- map()
- filter()
- reduce()
- some()
- every()
- flat()
- includes()
